# All baseline implementations for JML Specification Synthesis

- [Houdini](./houdini/): Classic, template based approach 
- [Daikon](./daikon/): Classic, execution trace based approach
- [SpecGen](./specgen/): Prompt based, spec mutation 
- [AutoSpec](./autospec/): Prompt based, program decomposing and static analysis
- [FormalBench](./formalbench/): Prompt based, advance prompting with repair guidance
